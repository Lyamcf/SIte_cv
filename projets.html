Mes Réalisations académiques

LA GOUTTE SUPRÊME

Le projet La goutte suprême est un projet web dont le but était de faire un site façade d’un magasin fictif. En groupe de trois, nous sommes parties sur quelque chose d’humoristique autour de bouteilles d’eau. On a donc codé en HTML et CSS ce site.

SPACE INVADERS

Le projet “Space invaders” a été réalisé à deux dans le but de reproduire une borne d’arcade et en symbolisant les boutons physiques avec des boutons sur l’écran. Ce projet Python m’a permis de pouvoir travailler en groupe et de découvrir la gestion d’objets. Ce projet m’a également montré la difficulté d’un cahier des charges et de la résolution de bugs.

GESTION DE PARTICULES

Le projet de Gestion de Particules est un projet à deux qui consiste à générer et à gérer le mouvement de différentes particules. Ce projet en Go implique d’optimiser son code pour éviter les bugs, ainsi que de bien commenter son code pour qu’il reste compréhensible de tous.
